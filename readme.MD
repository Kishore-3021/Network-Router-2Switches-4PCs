# IP Configuration Project: 1 Router, 2 Switches, 4 PCs

## Project Description

This is a basic computer networking project created in Cisco Packet Tracer.

The main goal of this project is to configure IP addresses, default gateways, and router interfaces so that PCs in two different LAN networks can communicate with each other.

The topology uses one router, two switches, and four PCs. PC1 and PC2 are connected to Switch 1. PC3 and PC4 are connected to Switch 2. Both switches are connected to the router.

---

## Network Topology

PC1 ─┐
     ├── Switch 1 ── Router ── Switch 2 ── PC3
PC2 ─┘                         └────────── PC4


Project Objectives

The objectives of this project are:

To create a basic network topology in Cisco Packet Tracer
To connect PCs with switches
To connect switches with a router
To configure IP addresses manually
To configure default gateways on PCs
To configure router interface IP addresses
To test communication between different LAN networks using ping

Skills Learned

Through this project, I learned:

How to create a basic network in Cisco Packet Tracer
How to connect routers, switches, and PCs
How to assign IP addresses manually
How to configure default gateways
How to configure router interfaces
How to use the no shutdown command
How to verify router interfaces
How to test network connectivity using ping
How communication happens between two different LAN networks

This project successfully demonstrates basic IP configuration and communication between two different LAN networks using one router.

PCs in LAN 1 and LAN 2 can communicate with each other because the router is configured with gateway IP addresses for both networks.
